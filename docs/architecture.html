  • MVP-1 ships with cascaded audio only; Gemini Live and NVIDIA remain future plugins.

  • Vertex EU embedding availability for ingest and call-time query embedding.

Data & knowledge layer

knowledge-ingestion
chunk → embed (Vertex EU) → pgvector upsert; hash-keyed re-index
PostgreSQL 16 + pgvector
tenant-scoped per-tenant indexes; 768-dim; warmed at session setup
embeddings
Vertex gemini-embedding-001 EU · same model: ingest + query
RAG scale path — one Postgres lineage: pgvector (Standard) → AlloyDB + ScaNN (Growth/Scale) → AlloyDB Omni (Sovereign). Same engine family → tier upgrades are lift-and-shift, migrated per-tenant on a RAM-headroom trigger (non-destructive: build parallel → parity eval → flip flag). Vertex RAG Engine dropped (extra engine family + sovereign-region hole).
Data boundary: ingestion chunks, vector rows, retrieval calls, governed audio, retention policy, and export jobs all require tenant_id; cross-tenant denial tests are mandatory.

Backend seam — swap the engine, not the car

MVP 1 · now

Cascaded

  • Deepgram + Claude + Cartesia
  • Frankfurt / EU, pay-per-use
  • The only MVP-1 backend
MVP 2

Gemini Live

  • Single native-audio model
  • Value: latency + residency
  • Blocked until audio interface, config store, and usage metering are archived
MVP 3

NVIDIA self-host

  • Riva ASR + Magpie TTS + LLM slot
  • Cassava SA / Kasi Lagos (sovereign GPU)
  • Gated: hard-residency tenant or volume inflection, after consent and Nemotron spike gates

Edge seams — vendor-agnostic channels & billing

MVP 1 · now → MVP 2

Widget ubiquity

  • Same widget.js everywhere: desktop & mobile web + in-app Android/iOS
  • All WebRTC (browser↔cloud); no phone network in MVP 1
  • WebView now, native SDK later — WebView weakens mic / echo-cancel / barge-in vs the p50 ≤1.5s budget; native is a planned upgrade, not an accident
MVP 2

Channel carriers (behind channel-ingress)

  • SIP / PSTN "dial a real number" + WhatsApp adapters
  • Telnyx — cost, Standard tier (US/EU HQ)
  • Africa's Talking — residency, Sovereign tier (in-region)
  • Carrier carries the call only — never the STT/LLM/TTS. Anti-fits: Vapi / Bland / Retell / Telnyx Voice AI bundle
MVP 1 · gap

Billing providers

  • Stripe can't onboard NG/KE merchants (2026)
  • billing_provider interface: Stripe (global) + Paystack/Flutterwave (Africa)
  • Tier entitlements provider-agnostic; only the charge/collect leg varies — verify recurring + metered on the African provider
